1.     Explique com suas palavras qual é a função do operador UNION.

Une as informações selecionadas exceto as repitidas

2.     Qual é a diferença entre UNION e UNION ALL?

UNION ALL une as informações selecionadas incluindo as repetidas enquanto o UNION não mostra as repetidas

3. Qual operador deve ser usado para encontrar registros que aparecem nas duas consultas?

a) UNION
b) UNION ALL
(X) INTERSECT

4. Observe a consulta abaixo e escreva qual será o resultado.

Resposta é Ana

2. Cláusula HAVING
Considere uma tabela chamada Entregas com as colunas: id_entrega, cidade e peso_entrega.

1. Complete: A cláusula WHERE é usada para filtrar Informações antes do agrupamento.

2. Complete: A cláusula HAVING é usada para filtrar Informações depois do GROUP BY.

3. Marque a consulta correta para mostrar cidades com peso total de entregas maior que 20.

a)

SELECT cidade, SUM(peso_entrega)
FROM Entregas
WHERE SUM(peso_entrega) > 20
GROUP BY cidade;

(X)

SELECT cidade, SUM(peso_entrega)
FROM Entregas
GROUP BY cidade
HAVING SUM(peso_entrega) > 20;

3.     Escreva uma consulta para mostrar o total de entregas por cidade.

Resposta:

SELECT cidade, COUNT(*)

FROM Entregas

GROUP BY cidade;

4.     Agora adapte a consulta anterior para mostrar apenas cidades com mais de 2 entregas.

Respossta:

SELECT cidade, COUNT(*)

FROM Entregas

GROUP BY cidade

HAVING COUNT(*) > 2;.

5.     Explique com suas palavras o que é uma subconsulta.

È uma consulta dentro da outra

6.     Crie uma consulta para mostrar clientes que fizeram pedidos acima de R$ 500.

Resposta:

SELECT nome

FROM Clientes

WHERE id_cliente IN (

    SELECT id_cliente

    FROM Pedidos

    WHERE valor_total > 500

);
