Quiz de fixacao
Qual comando no MySQL e utilizado para importar dados em massa a partir de um arquivo CSV?
Resposta: A) LOAD DATA INFILE
Qual e a funcao do comando FIELDS TERMINATED BY no MySQL?
Resposta: D) Determinar como os campos de um arquivo de dados sao separados.
Qual comando voce usaria no MySQL para exportar dados no formato CSV?
Resposta: C) SELECT INTO OUTFILE
Respostas curtas finais
O que e importar?
Trazer dados externos para o banco.

O que e exportar?
Salvar dados do banco em um arquivo externo.

Para que serve LOAD DATA INFILE?
Importar arquivos CSV para o MySQL.

Para que serve FIELDS TERMINATED BY?
Definir o separador dos campos.

Para que serve SELECT INTO OUTFILE?
Exportar dados do banco para um arquivo.

Qual foi a maior dificuldade encontrada na pratica?
Encontrar a pasta correta permitida pelo MySQL.


OBS: O MySQL Compiler não tem suporte para importar e exportar dados.
