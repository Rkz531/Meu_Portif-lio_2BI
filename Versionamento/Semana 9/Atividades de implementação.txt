6. Atividade D - Responda com suas palavras
1. Para que serve o GitHub Actions?
GitHub Actions serve para automatizar tarefas no GitHub, como testes, builds e deploys, sem precisar fazer tudo manualmente.

2. Por que não devemos colocar senhas diretamente no código?
Porque outras pessoas podem acessar essas senhas, causando problemas de segurança e vazamento de dados.
3. O que significa fazer rollback?
Rollback significa voltar o sistema para uma versão anterior quando a nova versão apresenta erros ou problemas.
4. Por que o Docker pode ajudar na entrega de software?
Porque o Docker empacota a aplicação com todas as dependências, fazendo ela funcionar da mesma forma em diferentes ambientes.
